# Используемое программное обеспечение

Эта инструкция перечисляет программы и сервисы, которые нужны для работы с учебным проектом.

---

# 1. Git

`Git` нужен для контроля версий.

С его помощью можно:

1. Сохранять изменения в проекте.
2. Смотреть историю изменений.
3. Создавать ветки для отдельных задач.
4. Отправлять код на GitHub.
5. Получать изменения от других участников.

Проверить установку Git можно командой:

```bash
git --version
```

---

# 2. Visual Studio Code

`Visual Studio Code` нужен как основной редактор кода.

В нем удобно:

1. Открывать папку проекта.
2. Редактировать файлы `.py` и `.md`.
3. Работать со встроенным терминалом.
4. Устанавливать расширения.
5. Запускать команды Git и Python.

Скачать VS Code можно с официального сайта:

```text
https://code.visualstudio.com/
```

---

# 3. GitHub

`GitHub` нужен для хранения проекта в интернете.

Он помогает:

1. Хранить удаленную копию репозитория.
2. Делиться кодом с другими участниками.
3. Смотреть изменения через сайт.
4. Создавать pull request.
5. Работать над проектом с разных компьютеров.

Открыть GitHub можно по адресу:

```text
https://github.com/
```

---

# 4. Python

`Python` нужен для запуска программ проекта.

С его помощью можно:

1. Запускать файлы `.py`.
2. Создавать виртуальное окружение `.venv`.
3. Устанавливать библиотеки.
4. Проверять работу учебных примеров.

Проверить установку Python можно командой:

```bash
python --version
```

Если команда не работает, можно попробовать:

```bash
python3 --version
```

---

# 5. AI-помощник

AI-помощник нужен для подсказок, объяснений и проверки идей.

Можно использовать любой доступный вариант:

1. ChatGPT.
2. GitHub Copilot.
3. Claude.
4. DeepSeek.
5. Gemini.

AI может помогать:

1. Объяснять ошибки.
2. Подсказывать команды.
3. Разбирать код простыми словами.
4. Помогать писать инструкции.
5. Предлагать варианты решения задач.

Важно: ответы AI нужно проверять. AI может ошибаться, поэтому финальное решение всегда должен понимать человек.

---

# 6. Краткая шпаргалка

Минимальный набор для работы:

```text
Git
Visual Studio Code
GitHub
Python
AI-помощник: ChatGPT / Copilot / Claude / DeepSeek / Gemini
```

Перед началом работы желательно проверить:

```bash
git --version
python --version
```
